tag de hiperlink

A taga hrefdefine um hiperlink(conexão), que é utilizado para vincular uma págia a outra

O atributo mais importante desta tag é o href que define a página de destino que sera exibida após o clique do mouse no link.

Por padrão, os links aparecerão da seguinte forma em todos os navegadores:

Um link não visitado é suilinhado em azul

um link visitado pe subilinhado em vermelho

um link ativo é subilinhado em vermelho

Exemplo

link para o arquivo h1-h6

clique aqui para visitar a página sobre marcadores h1-h6

Link para o site SENAI Suzano

Clique aquipara visitar a pagina a pagina do SENAI Suzano.
