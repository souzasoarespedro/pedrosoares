Listas não ordenadas

A tag ul (unordered list - lista não ordenada) representa uma lista de itens sem oredem rigida, por exemplo ingredientes em uma receita.

A tag li (list itens - da lista) representa um item que faz parte de uma lista.

      a tag ol (ordered list - lista ordenada) representa uma lista de itens numerados sequencialmente.

    • Primeiro item
    • Segundo item

    atributos

    reversed (ordem reversa)
    Exibe os itens da lista ordenada em ordem reversa

    1. item mais importante
    2. Item menos importante

    start (inicio)
    especifica o valor inicial para a numeração da lista.

    1. Item 5
    2. Item 6

    type (tipo)
    Indica o tipo de numeração;
    a - Indica letras minúscolas
    A - Indica letras maiúsculas
    i - indica algarismos romanos minúculos
    I - indica algarismos romanos maiuculos

    1. primeiro item
    2. segundo item
